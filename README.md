## MapReduce编程实例

#### 需求分析
1.需求：

统计每一个手机号耗费的总上行流量、下行流量、总流量

2.数据准备：

(1)输入数据格式：

时间戳、电话号码、基站的物理地址、访问网址的ip、网站域名、数据包、接包数、上行/传流量、下行/载流量、响应码
 
(2)最终输出的数据格式：

手机号码		上行流量        下行流量		总流量
